Stigla — design gallery

Static HTML/CSS replica of every app screen & state, for importing into Figma with the html.to.design Chrome plugin (it parses the page DOM into layers — Flutter web renders to canvas, so the real app is useless to it). One state = one page, iPhone 390×844 frame, dark theme, realistic Belgrade mock data. Semantic class names become layer names. This is a UI reference for the redesign, not production code.
