# Промпты задач

Постановки задач — по одному `.md` на задачу (напр. `fleet-id.md`). Сюда
кладётся спека/промпт ДО начала работы, чтобы новая сессия восстанавливала
контекст из файла, а не из чата.

⚠️ Репозиторий **публичный**. Не вставляй в промпты секреты, внутренние URL,
ID аккаунтов/ресурсов или детали доступа к источнику данных.
